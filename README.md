# CALSEES IT Project :-

A simple and responsive **Calculator Web Application** built as a practice project to perform basic arithmetic operations. This project demonstrates fundamental concepts of frontend development using HTML and CSS.

---

## Features

-  Addition  
-  Subtraction  
-  Multiplication  
-  Division  
-  Clear / Reset functionality  
-  Responsive design for all screen sizes  

---

## Technologies Used

- **HTML** – Structure of the calculator  
- **CSS** – Styling and layout.
